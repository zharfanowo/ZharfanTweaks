Tweaks Básicos

Nuestros tweaks básicos son el punto de partida perfecto para mejorar el rendimiento de tu PC. Estos ajustes son seguros y efectivos para cualquier usuario.

  • Deshabilite la protección de memoria DMA y el aislamiento de núcleos ("protección basada en virtualización").

    bcdedit /set vsmlaunchtype Off

    bcdedit /set vm No

    HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\FVE

    DisableExternalDMAUnderLock -> 0

    HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\DeviceGuard

    EnableVirtualizationBasedSecurity -> 0

    HVCIMATRequired -> 0

  • Utilice prioridad en tiempo real para la CPU.

    HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Image File Execution Options\csrss.exe\PerfOptions

    CpuPriorityClass -> 4

    IoPriority -> 3

  • Habilite la memoria administrada por kernel y deshabilite los parches Meltdown/Spectre.

    HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Session Manager\Memory Management

    FeatureSettings -> 1

    FeatureSettingsOverride -> 3

    FeatureSettingsOverrideMask -> 3

  • Deshabilite mitigaciones NTFS/ReFS adicionales.

    HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Session Manager

    ProtectionMode -> 0

  • Utilice el almacenamiento en caché de la memoria del sistema para mejorar el rendimiento.

    HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Session Manager\Memory Management

    LargeSystemCache -> 1

  • Desactivar el antivirus de Windows Defender(opcional)

    HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows Defender

    DisableAntiSpyware -> 1

    DisableRoutinelyTakingAction -> 1

    ServiceKeepAlive -> 0

  • Deshabilitar la compresión de RAM.

    powershell "Disable-MMAgent -MemoryCompression"

  • No permitir que los controladores se convinen en la memoria virtual.

    HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Session Manager\Memory Management

    DisablePagingExecutive -> 1

  • Deshabilite mitigaciones NTFS/ReFS adicionales.

    HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Session Manager

    ProtectionMode -> 0

  • Establezca una marca de tiempo confiable de 1 ms (como mínimo). Solo para sistemas sin modificaciones (se recomienda deshabilitarla con 0 en sistemas modificados).

    HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Reliability

    TimeStampInterval -> 1

  • Deshabilite la telemetría.

    HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\DataCollection

    AllowTelemetry -> 0

  • Deshabilitar la telemetría de la aplicación.

    HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\AppCompat

    AITEnable -> 0

  • Reducir el tamaño del búfer del ratón y del teclado.

    HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\mouclass\Parameters

    MouseDataQueueSize -> 16 decimal

    HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\kbdclass\Parameters

    KeyboardDataQueueSize -> 16 decimal

Si no te salen algunas de estas opciones en el editor de registro puedes crearla dandole click izquierdo, nuevo, y DWORD32BITS y ponerle los valores.
